## What is Marp?

Marp allows you to create beautiful slide presentations using simple Markdown syntax. It's perfect for developers who want to:

- Write presentations in their favorite text editor
- Version control their presentations with Git
- Export to multiple formats (HTML, PDF, PowerPoint)
- Include code snippets with syntax highlighting
- Use custom themes and styling
